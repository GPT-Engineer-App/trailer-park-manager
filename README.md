# trailer-park-manager

A yard management tool to track what trailers are assigned to what parking spaces as well as what orders are loaded onto the trailers. With the functionality to drag and drop  trailers to different spots as well as orders

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/trailer-park-manager.git
cd trailer-park-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
